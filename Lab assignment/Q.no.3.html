1. P Tag:

The < p> tag defines a paragraph.

syntax :

< p> This is some text in a paragraph < /p>


output:

This is some text in a paragraph

2. Underline Tag:

The < u> Tag define Underline Element

syntax:

< u> underline element < /u>


output:

underline element

3. Emphaszied Tag:

9. HTML Tag:

The < html> tag represents the root of an HTML document. The < html> tag is the container for all other HTML elements (except for the tag).

syntax:

< html lang="en">
< head>
< title>Title of the document< /title>
< /head>
< body>
< h4>This is a heading< /h4>
< p>This is a paragraph.< /p>
< /body>
< /html>


Output:
Title of the document

This is a heading

This is a paragraph.
